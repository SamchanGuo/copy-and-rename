# copy-and-rename
A Python tool for batch copying and renaming files on Windows.So,If you want to make multiple copies of a file and rename these files with different names, you can use this tool.

## Steps to use copy&rename.py:

Click the "Select Source File" button to choose the file to be copied.

Click the "Select List File" button to choose a txt file containing the filenames.

Click the "Select Output Directory" button to choose the save location.

When all three conditions are met, the "Start Processing" button will be automatically enabled.

Click the "Start Processing" button to begin batch copying and renaming.

Now the interface should be clearer, and the start button will appear and become enabled at the appropriate time.
